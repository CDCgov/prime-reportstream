# SFTP

Azure SSH key names determine SFTP instances and users
  * `pdh<env>-sftp01-lab1`
    * instance: `sftp01`
    * user: `lab1`
  * `pdh<env>-sftp01-lab2`
    * instance: `sftp01`
    * user: `lab2`
  * `pdh<env>-sftp01-lab1`
    * instance: `sftp02`
    * user: `lab1`

Seperate file shares exist for each instance/user combination

[Reference](https://github.com/atmoz/sftp)
