SettingCommand

abstract class SettingCommand(name: String, help: String) : CliktCommand
Content copied to clipboard

Base class to handle common stuff: authentication, calling, inputs and outputs. It has a composable set of methods for listing, getting, diffing, putting and deleting settings. The idea is to make concrete commands clear and concise by building primitives in this class.
