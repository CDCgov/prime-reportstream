Apply the mapper using the values from the current report item. Called after valueNames().

For example, if the schema had a mapper field defined for a minus mapper

  • name: some_element mapper: minus(x, 1)

apply would be called with an args list of "x", "1" and a values list of ElementAndValue(Element(x, ...), "9") where "9" is the value of the x element for the current item. The apply method would return "8", thus mapping the some_element value to the x value minus 1.
