class LIVDLookupMapper : Mapper
Content copied to clipboard

This is a lookup mapper specialized for the LIVD table. The LIVD table has multiple columns which could be used for lookup. Different senders send different information, so this mapper incorporates business logic to do this lookup based on the available information.

This function uses covid-19 schema elements in the following order:

  • device_id - From OBX-17.1 and OBX-17.3, may be a FDA GUDID or a textual description

  • equipment_model_id - From OBX-18.1, matches column 0

  • test_kit_name_id - matches column M

  • equipment_model_name - From STRAC, SimpleReport, and many CSVs, matches on column B

Example Usage

  • name: test_performed_system_version type: TABLE table: LIVD-SARS-CoV-2-2021-01-20 # Specific version of the LIVD table to use tableColumn: LOINC Version ID # Column in the table to map mapper: livdLookup()
