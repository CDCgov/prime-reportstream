The LookupSenderValuesetsMapper is used to lookup values from the "sender_valuesets" table/csv The args for the mapper are: args0 --> lookupColumn = the primary lookup field (usually "sender_id") args1 --> questionColumn = the secondary lookup field, expected to be the element name (i.e. patient_gender) The mapper uses the above arguments + the question's answer to retrieve a row from the table
