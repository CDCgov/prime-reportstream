The set of filtering objects available per Organization or per Receiver (and someday, per Sender too?)

Examples of FilterTypes that can be applied:

Parameters

jurisdictionalFilter

  • used to limit the data received or sent by geographical region

qualityFilter

  • used to limit the data received or sent, by quality

routingFilter

  • used to limit the data received or sent, by who sent it.

processingModeFilter

  • used to limit the data received to be either "Training", "Debug", or "Production" We allow a different set of filters per topic
