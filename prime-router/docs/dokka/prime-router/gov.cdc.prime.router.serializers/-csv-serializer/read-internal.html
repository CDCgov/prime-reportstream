Reads an internal format document (one that is in CSV, but matching the internal "kitchen sink" schema) and returns the Report object for the file.

Parameters

useDefaultsForMissing

if a field is missing that the Report object expects, this instructs the function to use the default value provided by the schema (or empty string) if it doesn't exist. This was added because some older files might need to be reprocessed, and this allows that to happen without error due to a missing column
