# Using GitHub Codespaces

> GitHub codespaces provide a browser-only dev environment.

## :construction: *Documentation in-progress* :construction:

 1. Either be approved by [CDCgov](https://github.com/CDCgovD) for codespace access, or sign-up for the [free codespaces beta](https://docs.github.com/en/codespaces/managing-your-codespaces/managing-gpg-verification-for-codespaces) (weeks to months for approval)
 2. Go to repo [homepage in GitHub](https://github.com/CDCgov/prime-reportstream)
 3. Click on **<> Code**
 4. Select **Codespaces** tab
 5. Click **Create codespace on master** (or another branch if chosen)
 6. Wait for environment to finish provisioning.
    * This step includes starting the VM and running docker containers.

## Tips

1. To sign a commit, [use the `-S` flag](https://docs.github.com/en/codespaces/managing-your-codespaces/managing-gpg-verification-for-codespaces)
